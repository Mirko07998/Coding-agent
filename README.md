# Hello World Module

This repository contains a simple Python module that prints "Hello, World!" to the console.

## Files

- `hello_world.py`: The main module that includes the `print_hello_world` function.
- `test_hello_world.py`: Unit tests for the `hello_world` module.

## Usage

To run the module, execute the following command in the terminal:

```bash
python hello_world.py
```

This will output: 
```
Hello, World!
```

## Testing

To run the tests, execute the following command:

```bash
python -m unittest test_hello_world.py
```

This will run the unit tests and verify the functionality of the module.
